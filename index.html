Sunset

CLI client untuk layanan internet mobile Indonesia tertentu. Akses dan kelola paket langsung dari terminal Termux kamu.

🚀 Cara Install 📢 Telegram Channel
Langkah Pertama
Dapatkan Environment Variables

Sebelum menjalankan script, kamu perlu mendapatkan environment variables terlebih dahulu. Kunjungi channel Telegram kami dan salin variabel yang tersedia.

✈️ Buka Channel Telegram
# Setelah mendapat variabel, buat file .env di folder me-cli-sunset
# Contoh isi file .env:
TOKEN=your_token_here
# dst...
Instalasi Termux
Cara Menjalankan
1

Update & Upgrade Termux

pkg update && pkg upgrade -y
2

Install Git

pkg install git -y
3

Clone Repository

git clone https://github.com/purplemashu/me-cli-sunset
4

Masuk ke Folder

cd me-cli-sunset
5

Jalankan Setup

bash setup.sh
6

Jalankan Script

python main.py
